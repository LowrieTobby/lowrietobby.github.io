表格对齐方式方法：

  • 左对齐： |:——|
  • 右对齐： |——:|
  • 居中对齐： |:—-:|

图片&链接

链接

Markdown中插入链接可以使用 [链接名称](url"描述") 的形式来插入链接，url 可以是http链接或者是本地文件路径

Blog.lowrie.club
点击这里这里进入博客主页

链接也可以写成脚注的形式

进入主页
点击后面的链接以打开网站链接

图片

形式和链接相同，只不过在前面加上 ! 符号

This is a 16x16 Lowrie icon

Markdown插入的图片不能指定图片尺寸，但可以通过使用html标签 <img> 来自定义插入的图片
