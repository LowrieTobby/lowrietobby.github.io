列表迭代器 ListIterator

LisrIterator (列表迭代器) 是 Iterator (迭代器) 的一个子接口，是List集合的

列表迭代器的出现突破了迭代器的局限性，迭代器除了可以进行判断 hasNext() 和取出next()操作之外，只能进行对集合元素的删除 remove(Object o)。而列表迭代器除了含有迭代器的所有方法以外，还增添了专门针对列表集合元素操作的增、改、查以及其他方法

方法

返回类型 方法与方法描述
void add(E e)将指定的元素插入列表（可选操作）。
boolean hasNext()返回 true如果遍历正向列表，列表迭代器有多个元素。
boolean hasPrevious()返回 true如果遍历反向列表，列表迭代器有多个元素。
E next()返回列表中的下一个元素，并且前进光标位置。
int nextIndex()返回随后调用 next()返回的元素的索引。
E previous()返回列表中的上一个元素，并向后移动光标位置。
int previousIndex()返回由后续调用 previous()返回的元素的索引。
void remove()从列表中删除由 next()或 previous()返回的最后一个元素（可选操作）。
void set(E e)用 指定的元素替换由 next()或 previous()返回的最后一个元素（可选操作）。

为什么使用

除了上述所说的比 iterator 多出的优势以外，使用 ListIterator 可以避免在迭代过程中因使用集合的方法修改元素而出现的 ConcurrentModificationException 异常。
